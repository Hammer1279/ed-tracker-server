ED-Server
==========

> This Server supports both `ED-MFD` by `@fightingdoggo` and `ed-tracker` by `@Hammer1279`
> You can connect either via `Rest-API` or `WebSocket`
---

Installation
------------

```xl
gh repo clone Hammer1279/ed-server
```
